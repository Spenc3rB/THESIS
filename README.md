## About

This repository contains code and documentation related to my thesis work in Systems Engineering at Colorado State University. The focus of the thesis is on using SysML as a reconstruction framework for cyber-physical systems, especially in the context of cybersecurity vulnerability management.

> See [CWE](https://cwe.mitre.org/), [CAPEC](https://capec.mitre.org/), and [CVE](https://cve.mitre.org/).

## Contributions

- Documentation of the methodology and findings of the thesis.
- Scripts and tools developed during the research.
- SysML profile for Cybersecurity Vulnerability Management Systems (CVMS):
  - Custom: CVE and CWE stereotypes and traceability relationships.
- Case Studies applying the CVMS profile to SysML as a reverse engineering framework.

## Repository Structure

- `case-studies/`: Contains detailed case studies applying the SysML CVMS profile.
- `code/`: Contains scripts and tools developed for data conversion and processing.
- `media/`: Contains images and media files used in the documentation.
- `SysML/`: Contains the SysML profile and related case studies.
- `schedule.md`: Living document for thesis deadlines and schedule.
